コンソール出力に色を付ける。

16進数6桁のRGBコードを用いて、色を指定し、それについての文字列を取得します。

例: 白の場合 -> "#FFFFFF"

例: 色名の場合 -> "white"

Parameters
input_色を付けたい対象の文字列
color_色コード(#rrggbb)、または、表の色名のいずれかを指定します。

Returns
color_をunsignedに変換したものをconvertedとしたとき、戻り値は以下のようになります。
return Color(input_, converted);
正しく指定されていない場合は、"[PrintHelper::Color] error: 色コードが不正です。({input_}, {color_})"と返します。
Note
Truecolorが使用できない場合は、正しく表示されない場合があります。
