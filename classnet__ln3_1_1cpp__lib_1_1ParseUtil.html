static std::vector< std::string > net_ln3::cpp_lib::ParseUtil::split ( const std::string &  str,
const std::string &  delim = " " 
)
static

delimを基準に分割し、結果の配列を返す。

出力にdelimは含まれません。

delimを基準に文字が存在するかを問わず配列化するため、以下のような入力になります。

delimは「,」とする。",a,,b,"を入力した場合{"", "a", "", "b", ""}となる。

strに空文字列を与えた場合、空文字列のみを含む配列を返します。

これは入力(str)にdelimが含まれない場合の動作に等しいです。

Parameters
str分割対象の文字列
delim分割の基準となる文字列
Returns
delim.empty(): 1文字単位で分割され、文字列に変換された配列
else: delimを基準に分割された文字列の配列
